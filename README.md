Breakout Retro adalah game brick breaker klasik dengan tampilan retro sederhana.
Dibangun menggunakan Python + Pygame, game ini mendukung:
Main menu interaktif (keyboard & mouse)
Countdown 5 detik sebelum game mulai
Paddle yang smooth
Bola dengan fisika pantulan realistis
Blok warna-warni model Famicom
Power-up random drop
Efek Game Over
Cursor otomatis hilang saat bermain (immersive)
Ringan, simple, dan cocok buat tugas kuliah ataupun portfolio.
